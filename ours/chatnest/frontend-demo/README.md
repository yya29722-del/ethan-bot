# AI 聊天前端演示版

这是一个静态前端演示包。它使用假数据和模拟流式事件，不需要后端、API key 或模型账号。

## 占位符说明

这个公开演示版有意使用通用占位图标、系统字体和中性文案。它只是占位符版本。要替换 logo、图标和字体，请从下面链接下载单独素材包，并按 `BRANDING.md` 操作：

https://drive.google.com/drive/folders/1EFaL-cwFn262Mu8L9s-cO9dw6FC4LAMr

## 运行

可以直接打开 `index.html`，也可以在这个目录里启动任意静态文件服务器：

```bash
python3 -m http.server 8080
```

然后打开 `http://127.0.0.1:8080/`。

这个公开包不包含供应商 logo、专有字体、私人 prompt 或私有品牌素材。
